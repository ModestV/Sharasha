# <font color="#edff21">4. Понятие последовательности. Способы задания, предел. Свойства последовательности (ограниченность, монотонность). Теоремы о пределах. Теорема Вейерштрассе. Неограниченно большая последовательность. </font> 

Последовательность - взаимно-однозначное соответствие между натуральными числами некоему множеству действительных чисел X с элементами xn. 

Как задаётся последовательность:
1. Перечислением
2. Общим членом
3. Рекуррентным соотношением

Число а называется пределом последовательности, если выполняется равенство в |xn - a| < E для любого положительного Е найдётся такой номер N, что равенство выполняется  для N и всех номеров после N.

Свойства:
1. Последовательность считается ограниченной, если для любого n in N выполняется условие |xn| < M, где M существующее положительное число.
2. Последовательность называется монотонной, если для любого n in N выполняется: x(n+1) >= xn либо наоборот

Теоремы о пределах:
Если последовательности yn и xn имеют пределы b и a, то
1. lim n->inf (yn+xn) = a + b
2. lim n-> inf(yn•xn) = a •b
3. если b!=0, то lim n->inf (yn/xn) = a/b

Теорема о двух полицейских:
если есть последовательности xn yn zn, выполняется xn < yn < zn, а xn и zn имеют предел а, то yn тоже имеет предел a.
	Пример: 
	$y_n = \frac{cos({n^2})}{n^2}$
	$x_n = -\frac{1}{n^2}$    $\lim\limits_{n\to\infty}x_n=0$
	$z_n = \frac{1}{n^2}$       $\lim\limits_{n\to\infty}z_n=0$
	$x_n<y_n<z_n\implies \lim\limits_{n\to\infty}y_n=0$

Теорема Вейерштрасса.
Любая монотонная ограниченная последовательность имеет предел.
	$x_n = 1 -\frac{1}{2^n}$

Неограниченно большая последовательность - последовательность, чей модуль постоянно растёт.
