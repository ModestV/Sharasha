#Объяснение
Ну в общем благодарю *кое-кому* теперь сюда автоматически коммитяться все-превсе приколы, которые я пишу у себя в обсидиане, пока учусь в УрФУ.

#Использование
Просмотр содержимого на GitHub возможен, но не особо полезен, поскольку рендериться в обсидиане и здесь немножко иначе. Рекомендую использовать Obsidian с плагином Excildraw, чтобы иметь возможность просматривать файлы практик.
